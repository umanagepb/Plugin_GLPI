Teste da Integração Clockify

Esta página simula a estrutura de um ticket do GLPI para testar a integração com o Clockify.

Teste 1: GLPI 10.x (Bootstrap Layout)

Ticket #12345 - Problema no sistema

O botão deve aparecer no cabeçalho do card, à direita do título.

Teste 2: GLPI 9.x (Table Layout)

Ticket #67890 - Solicitação de acesso
O botão deve aparecer dentro do cabeçalho da tabela, alinhado à direita.

Teste 3: Formulário com campos hidden

O botão deve capturar o ID do campo hidden e o título do input.

Instruções para teste:

  1. Abra o console do navegador (F12)
  2. Procure por mensagens do "Clockify" nos logs
  3. Verifique se os botões aparecem nos locais corretos
  4. Clique nos botões para testar a funcionalidade (gerará erro de API por usar credenciais de teste)

Configuração para uso real:

  1. Configure sua API Key real do Clockify em window.clockifyIntegrationConfig.apiKey
  2. Configure seu Workspace ID real em window.clockifyIntegrationConfig.workspaceId
  3. Teste em uma página de ticket real do GLPI
